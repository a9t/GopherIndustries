# GopherIndustries
This is a simple terminal game inspired by Factorio. You automate resource extraction, item creation and their assembly in increasingly comlex entities.

## How To
Retrieve project:
```
go get github.com/a9t/GopherIndustries
```

Build:
```
go build
```

Execute:
```
./GopherIndustries
```
